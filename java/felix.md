Felix Framework
===============

Download zip archive:

    $ firefox http://felix.apache.org/downloads.cgi

Unzip the archive:

    $ unzip org.apache.felix.main.distribution-4.2.1.zip

Run the framework:

    $ cd felix-framework-4.2.1
    $ java -jar bin/felix.jar
